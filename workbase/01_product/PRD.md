# PRD — GEUMTAE Export (금태 프리미엄 수출 서비스)

## 1. 문제 정의

**해결하려는 문제**: 일본 고급 레스토랑 식자재 바이어가 품질·납기 안정성이 보장된 한국산 프리미엄 활·선어 소싱처를 찾지 못하는 문제.

**타겟 사용자**: [[user_stories]] 참조 (타나카 켄지 — 도쿄 오마카세 조달 담당 / 야마모토 사토시 — 오사카 호텔 구매 매니저)

**왜 지금인가**:
- 일본산 참돔: 중국 수입 금지(2023~)로 공급 불안, 브랜드 포화
- 노르웨이 연어: 파인다이닝 차별화 불가
- 기존 수협 채널: 금태 전문 공급 거의 없음, 서류·정보 지원 無

## 2. 목표 (Goals)

### Must (MVP 필수)

| 목표 | 수치 기준 | 측정 시점 |
|---|---|---|
| 납기 준수율 | 95% 이상 (도착일 ±1일) | 운영 3개월차 |
| 선도 불량 반품률 | 2% 미만 | 운영 3개월차 |
| 계약 바이어 수 | 10개 레스토랑 이상 | 1년차 |
| 재주문율 | 70% 이상 (3개월 이내) | 1년차 |

### Should

| 목표 | 수치 기준 |
|---|---|
| 연간 출하량 | 50톤 |
| 연간 매출 | $108만~$150만 |
| 샘플 → 계약 전환율 | 50% 이상 |

## 4. 핵심 기능 (Features)

### FR-001: HACCP 인증 완비 및 서류 패키지
- **설명**: 국내 HACCP 인증 유지, 납품마다 위생증명서·원산지증명서·HACCP 로트번호 동봉
- **OS 점수**: JTBD 핵심 성패 변수 (정성 검증)
- 수락 기준:
  - [ ] 납품 100%에 HACCP 인증서 사본 동봉
  - [ ] 일본 통관 거부 0건

### FR-002: 납기 SLA 계약 관리
- **설명**: 도착일 ±1일 보장 + 지연 보상 조건 계약. 납품별 SLA 이행 현황 기록
- **OS 점수**: OS = 16 (납기 지연 최소화)
- 수락 기준:
  - [ ] 납품 95% 이상 약정일 ±1일 이내 도착
  - [ ] SLA 위반 건 보상 자동 산정

### FR-003: 냉장 콜드체인 운영
- **설명**: 출하~도착 0~5°C 유지. IoT 센서로 온도 이탈 즉시 감지
- **OS 점수**: OS = 16 (선도 불량 최소화)
- 수락 기준:
  - [ ] 온도 이탈(5°C 초과) 발생률 1% 미만
  - [ ] 온도 로그 납품 100% 기록

### FR-004: 실시간 물류 추적
- **설명**: 발주 확정 후 바이어에게 추적 링크 자동 발송. 단계별 상태 표시
- **OS 점수**: OS = 14
- 수락 기준:
  - [ ] 납품 100% 추적 링크 제공
  - [ ] 상태 갱신 30분 이내 반영

### FR-005: 통관 서류 원스톱 대행
- **설명**: 위생증명서, 원산지증명서, 수출신고필증 등 전량 사전 준비 및 제출 대행
- **OS 점수**: OS = 15 (서류 처리 시간 단축)
- 수락 기준:
  - [ ] 바이어 직접 처리 서류 0건
  - [ ] 통관 서류 오류 지연 0건

### FR-006: 일본어 전담 고객 창구
- **설명**: 바이어별 전담 담당자 1인. LINE/KakaoTalk 일본어 소통. 발주·문의·클레임 일원화
- 수락 기준:
  - [ ] 메시지 24시간 이내 응답률 95%
  - [ ] CS 만족도 4/5점 이상 (분기별 조사)

### FR-007: 도착 불량 교환 보증
- **설명**: 도착 선도 기준 미달 시 재납품 또는 전액 환불. 클레임 48시간 이내 처리
- **OS 점수**: OS = 16
- 수락 기준:
  - [ ] 클레임 100% 48시간 이내 처리
  - [ ] 반품률 2% 미만 유지

### FR-008: 무료 샘플 프로그램
- **설명**: 신규 바이어 2kg 무료 샘플 (콜드체인 적용, 선도 보증, 일본어 조리 가이드 동봉)
- 수락 기준:
  - [ ] 발송 2주 이내 피드백 수집률 100%
  - [ ] 계약 전환율 50% 이상 목표

## 5. 범위 밖 (Out of Scope)

MVP에 포함하지 않는 것:
- B2C 플랫폼 및 일반 소비자 직판 채널
- 가공식품 개발 (통조림, 훈제, 냉동 가공품)
- 자사 선박·운송 수단 운용
- 일본 현지 법인 설립 (단기 MVP 기간 내)
- 가격 경쟁 전략 (JTBD 검증에서 기각)
- ASC 국제 양식 인증 취득
