## 1. 페르소나 스펙트럼

### Early Adopter — "탐색형 조달 담당자"

| 항목 | 내용 |
|---|---|
| 이름 | 타나카 켄지 (田中健司) |
| 역할 | 도쿄 미쉐린 2스타 일식당 식자재 조달 담당 |
| 나이 | 38세 |
| 회사 | 도쿄 긴자 소재 오마카세 레스토랑, 연매출 약 3억 엔 |

**구매 행동**
- 매 시즌 신규 어종 2~3개 소싱 시도
- 한국·유럽 등 수입산 활어 실험적으로 도입한 경험 있음
- 공급자와 직접 방문 관계 구축 선호

**Pain Point**
- 기존 국내산 참돔 단가 상승 + 어획량 불안정
- 외국산 신규 어종 도입 시 식품안전 서류 확인 번거로움
- 콜드체인 불안정으로 품질 편차 경험

**전환 트리거**
- 품질 일관성 보장 샘플 제공 + HACCP 서류 완비
- 국내 경쟁 레스토랑의 금태 메뉴 런칭 소식
- 한국 현지 방문 팸투어 참여

### Mainstream — "안정 추구형 구매 매니저"

| 항목 | 내용 |
|---|---|
| 이름 | 야마모토 사토시 (山本聡) |
| 역할 | 오사카 호텔 체인 F&B 구매 매니저 |
| 나이 | 46세 |
| 회사 | 오사카 5성급 호텔 체인, 연간 수산물 구매 예산 약 5억 엔 |

**구매 행동**
- 기존 승인된 공급업체 리스트(Approved Vendor List) 중심 구매
- 신규 공급자 도입 시 내부 심의 6개월 이상
- 가격보다 납기·품질 안정성 우선

**Pain Point**
- 내부 심사 기준(위생증명서·검역서류) 충족 공급자 부족
- 기존 에이전시 마진이 높아 가격 경쟁력 약화
- 신규 어종 도입 후 직원 조리 교육 부담

**전환 트리거**
- 기존 에이전시가 금태를 취급 시작할 경우 자동 검토
- 레퍼런스 레스토랑 3개 이상 납품 실적 확인 후 시범 발주
- 인증서 일체 + 조리 레시피 가이드 제공

### Resistant — "리스크 회피형 총주방장"

| 항목 | 내용 |
|---|---|
| 이름 | 스즈키 히로시 (鈴木浩) |
| 역할 | 교토 전통 가이세키 료칸 총주방장 |
| 나이 | 57세 |
| 회사 | 교토 100년 전통 료칸, 연간 수산물 구매 약 1억 엔 |

**구매 행동**
- 20년 거래 국내 어판장 중심, 수입산 거의 없음
- 신규 어종 도입 = 메뉴 철학 훼손으로 인식
- 가격 협상보다 관계·신뢰 중시

**Pain Point**
- 명시적 Pain 없음 — 현 시스템에 만족
- 외국산에 대한 고객 인식 우려 ("국내산 고집" 스토리 포기 못함)

**전환 트리거 (매우 제한적)**
- 일본 내 유명 셰프가 금태를 사용하고 미디어 노출
- 교토 현지 식자재 에이전시의 강력한 추천
- 단기적 접근 불필요 — 3년차 이후 검토

## 2. 고객 여정 지도 (CJM) — 일본 고급 레스토랑 바이어 기준 (타나카 켄지)

### 단계별 분석

#### 1단계: 인지 (Awareness)

| 항목 | 내용 |
|---|---|
| 행동 | 수산물 업계 뉴스레터 구독 / 식자재 박람회 참석 / 동료 셰프 SNS 팔로우 |
| 감정 | 3/5 (관심 낮음, 무관심 상태) |
| Pain | 금태라는 어종 자체를 모름 / 한국산 양식어류 = 광어·넙치 인식 고정 |
| 접촉점 | 식품박람회 부스, 업계 잡지 광고, 셰프 커뮤니티 |
| 기회 | 박람회 시식 이벤트 + 명함 교환, 셰프 인플루언서 활용 |

#### 2단계: 탐색 (Consideration)

| 항목 | 내용 |
|---|---|
| 행동 | 공급사 웹사이트 검색 / 에이전시에 금태 소싱 문의 / 사진·영상 자료 검토 |
| 감정 | 3.5/5 (궁금하지만 신뢰 부족) |
| Pain | 한국어 자료만 존재 / 가격·최소 주문량 정보 불명확 / 인증서 보유 여부 불확실 |
| 접촉점 | 웹사이트, 에이전시 연락, 이메일 자료 요청 |
| 기회 | 일본어 제품 카탈로그 + 인증서 사본 즉시 제공 체계 |

#### 3단계: 평가 (Evaluation)

| 항목 | 내용 |
|---|---|
| 행동 | 샘플 요청 / 조리 테스트 / 경쟁 제품(일본산 참돔·제주 넙치)과 비교 |
| 감정 | 4/5 (기대와 불안 공존) |
| Pain | 샘플 배송 중 선도 저하 / 가격 대비 품질 판단 기준 없음 / 결정 전 상사 설득 부담 |
| 접촉점 | 샘플 박스, 전화/화상 미팅, 현지 에이전시 동석 미팅 |
| 기회 | 전용 샘플 키트(조리 가이드 + 영양 성분표 + 비교 시식 체크리스트 포함) |

#### 4단계: 구매 (Purchase)

| 항목 | 내용 |
|---|---|
| 행동 | 시범 발주 (소량) / 계약서·검역 서류 검토 / 결제 조건 협의 |
| 감정 | 3.5/5 (기대감 + 서류 처리 피로) |
| Pain | 수입 서류 복잡성(위생증명서·원산지증명서 등) / 납기 불확실성 / 소량 발주 시 배송비 부담 |
| 접촉점 | 계약서, 물류 에이전시, 세관 서류 |
| 기회 | 서류 패키지 선제 제공 + 소량 발주 무료 배송 정책(첫 3회) |

#### 5단계: 사용 (Use)

| 항목 | 내용 |
|---|---|
| 행동 | 메뉴 개발 / 고객 반응 관찰 / 재고 소진 속도 체크 |
| 감정 | 4.5/5 (품질 만족 시) / 2/5 (품질 편차 발생 시) |
| Pain | 금태 조리법 숙지 부족 (한국 전통 어종이라 조리 정보 적음) / 품질 편차 시 컴플레인 경로 불명확 |
| 접촉점 | 제품 자체, 조리 가이드, A/S 연락처 |
| 기회 | 한국 요리사 화상 조리 시연 세션 + 클레임 24시간 응대 보장 |

#### 6단계: 재구매 (Repurchase)

| 항목 | 내용 |
|---|---|
| 행동 | 정기 발주 전환 / 다른 셰프 추천 / 소셜 미디어 언급 |
| 감정 | 5/5 (신뢰 구축 완료 시) |
| Pain | 수요 급증 시 납기 지연 우려 / 가격 인상 가능성 불안 |
| 접촉점 | 담당 영업, 정기 뉴스레터, 신제품 시즌 공지 |
| 기회 | 연간 계약(Annual Contract) + 가격 고정 조건 제공, VIP 고객 先출하 보장 |

## 3. CJM 인사이트 기반 Value Proposition 초안

### Pain이 가장 강한 접촉점 2개

**1위: 평가 단계(3단계) — 샘플 품질 및 정보 부족**
- 감정 저점: 기대와 불안 공존
- 핵심 Pain: 샘플 배송 중 선도 저하, 비교 판단 기준 없음

**해소 방안:**
- 24시간 이내 활어→냉장 가공 + 급속 냉동 전용 샘플 키트 개발
- 일본어 조리 가이드 + 참돔 대비 금태 특성 비교 자료 동봉
- 현지 에이전시 동석 방문으로 신뢰 접점 확대

**2위: 구매 단계(4단계) — 서류·물류 복잡성**
- 감정 저점: 설레임이 행정 피로로 전환
- 핵심 Pain: 위생증명서·원산지증명서 등 수입 서류 수작업 처리 부담

**해소 방안:**
- 수입 서류 전자 패키지화 (PDF 자동 생성 + 영문/일문 번역본 포함)
- 통관 대행 에이전시 연계 서비스 패키지 제공
- 소량(50kg 미만) 첫 3회 배송비 무료 정책

### VP 초안 (Gain / Pain / Job 구조)

| 구분 | 내용 |
|---|---|
| Job (해야 할 일) | 안정적으로 고품질 한국산 금태를 납기 확실하게 소싱하고, 레스토랑 메뉴 경쟁력을 유지한다 |
| Pain (제거할 고통) | 인증 불확실, 샘플 선도 저하, 수입 서류 번거로움, 납기 불안정 |
| Gain (제공할 이득) | HACCP 완비 서류 즉시 제공 + 전용 샘플 키트 + 정기 납기 보장 계약 |

**VP 문장 (초안):**
"금태 전문 공급자는 HACCP 인증 서류부터 일본어 조리 가이드, 정기 납기 보장 계약까지 — 바이어가 '구매 결정'에만 집중할 수 있도록 모든 조달 장벽을 선제 제거합니다."
