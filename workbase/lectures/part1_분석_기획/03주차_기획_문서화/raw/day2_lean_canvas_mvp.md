## 1. Lean Canvas

### 블록 구성 개요

Lean Canvas는 9개 블록으로 비즈니스 모델의 핵심 가설을 한 장에 압축한다.
아래 각 블록은 1~2주차 분석 결과(Porter's 5F, SWOT, TAM-SAM-SOM, 페르소나, JTBD)를 직접 반영한다.

---

### 블록 1: Problem (상위 3개)

1. **선도 불량**: 한국산 수산물이 일본 도착 시 선도가 저하되어 고급 코스 메뉴 사용 불가
2. **납기 불안**: 수입 수산물의 도착일 예측 불가, 긴급 대체 소싱 비용 및 메뉴 변경 리스크 발생
3. **서류·통관 복잡성**: 위생증명서, 원산지증명서 등 일본 수입 요건 서류를 바이어가 직접 처리해야 하는 행정 부담

**기존 대안**
- 도요스 도매시장: 신선도 높으나 한국산 특정 어종 구하기 어려움
- 기존 수산 브로커: 납기·품질 일관성 보증 없음, 일본어 지원 부재

---

### 블록 2: Customer Segments

**Early Adopter (초기 집중 타겟)**
- 도쿄 오마카세·파인다이닝 레스토랑 식자재 담당자
- 연간 수산물 구매 1억엔 이상
- 독자적 소싱 루트 구축에 관심 있는 셰프-오너

**향후 확장 세그먼트**
- 오사카·나고야 고급 일식 레스토랑
- 싱가포르·방콕·하노이 프리미엄 호텔 F&B 담당자
- 일본 식자재 수입 도매상 (유통 파트너)

---

### 블록 3: Unique Value Proposition

> "납기 SLA 계약 + HACCP 완비 + 일본어 전담 창구를 패키지로 제공하는 한국산 금태 수출 서비스.
> 기존 수산 브로커 대비 품질 일관성 보증과 투명한 콜드체인 추적으로 조달 리스크를 제거한다."

**핵심 차별점 한 줄**
> "품질과 납기를 계약으로 보증하는 수산물 공급처"

---

### 블록 4: Solution

| 문제 | 솔루션 |
|------|--------|
| 선도 불량 | 콜드체인 IoT 온도 모니터링 + 도착 불량 전량 교환 보증 |
| 납기 불안 | 납기 SLA 계약서 (도착일 ±1일 보장, 지연 보상 명시) |
| 서류 복잡성 | 통관 서류 원스톱 대행 + 일본어 전담 담당자 (LINE/KakaoTalk) |

**추가 솔루션**
- 무료 샘플 프로그램: 초기 2kg 샘플 제공, 선도 보증
- 일본어 조리 가이드: 규격별 손질법·조리 추천 (PDF 제공)
- 정기 발주 계약: 주 1~2회 납품 사이클, 수량 예약제

---

### 블록 5: Channels

**인지 (Awareness)**
- 도쿄 식자재 전시회 참가 (FOODEX JAPAN 등)
- 일본 식품 수입 에이전트·브로커 소개 경로
- 한국 수산물 수출 지원 기관 연계 (aT 한국농수산식품유통공사)

**전환 (Conversion)**
- 무료 샘플 프로그램 → 품질 검증 후 계약
- 도쿄 현지 미팅·시식 행사

**유지 (Retention)**
- 전담 담당자 LINE 채널 (정기 소통)
- 월간 품질 리포트 + 납기 실적 보고서 제공

---

### 블록 6: Revenue Streams

| 항목 | 단가 | 예상 물량 | 연간 매출 |
|------|------|-----------|-----------|
| 금태 선어 (1kg당) | $22~30 (시장가 기준) | 1년차 50톤 | $110만~$150만 |
| 납기 SLA 프리미엄 | 기본가 +10~15% | 전체 물량 | SLA 계약 포함 |
| 샘플 프로그램 | 무료 (고객 획득 비용) | - | - |

**수익 모델 구조**
- 직납 계약: 레스토랑과 연간 공급 계약, 단가 고정
- 스팟 주문: 계약 외 추가 발주 (단가 프리미엄 +5%)

**1년차 목표**: 50톤 / $108만~$150만 (SOM 기반)

### 블록 7: Cost Structure

| 비용 항목 | 설명 | 비고 |
|-----------|------|------|
| 수산물 원가 | 금태 양식장 매입가 | 전체 비용의 가장 큰 비중 |
| 콜드체인 물류 | 냉장 컨테이너, 항공/해상 운임, 국내 냉장 창고 | 국제 운송비 포함 |
| HACCP 인증 유지 | 국내 HACCP 연간 갱신, 일본 수입 요건 적합 서류 | 초기 인증 취득 비용 별도 |
| 통관 대행 | 일본 통관 에이전트 수수료 | 건당 or 월 정액 |
| 인건비 | 일본어 전담 CS 담당자, 영업 담당자 | 최소 2~3인 |
| 샘플 비용 | 초기 바이어 유치용 무료 샘플 | 고객 획득 비용 처리 |
| 플랫폼·시스템 | 물류 추적 API, 발주 관리 시스템, IoT 장비 | 클라우드 기반 |

### 블록 8: Key Metrics

| 지표 | 목표 (1년차) |
|------|-------------|
| 납기 준수율 | 95% 이상 (±1일 기준) |
| 선도 불량 반품률 | 2% 미만 |
| 계약 바이어 수 | 10개 레스토랑 |
| 재주문율 | 70% 이상 (3개월 이내) |
| 월 출하량 | 4톤 (연 50톤 목표) |
| 평균 CS 응대 시간 | 24시간 이내 |

---

### 블록 9: Unfair Advantage

| 우위 항목 | 내용 |
|-----------|------|
| 어종 희소성 | 일본 시장 내 한국산 금태 공급처 자체가 희소 |
| 콜드체인 노하우 | 한국-일본 수산물 콜드체인 구축 경험 (모방 시간 필요) |
| 양식장 직계약 | 중간 브로커 없이 양식장과 직접 계약, 원가 경쟁력 |
| 한국어·일본어 이중 대응 | 양쪽 언어 전담 담당자 보유 (단기간 모방 어려움) |
| KSF 선점 | HACCP + 납기 SLA + 일본어 창구 3종 세트를 동시 보유한 경쟁자 부재 |

## 2. 비즈니스 모델 가설 검증 설계

### 검증 원칙

- 가설은 가장 리스크가 큰 가정(Leap-of-Faith Assumption)을 우선 검증한다.
- JTBD 분석에서 기각된 가설(어종 희소성·가격 경쟁력)은 이미 제외.

---

### 가설 1: 납기 SLA 계약이 바이어 전환 결정 요인이다

**가설 내용**
일본 오마카세 레스토랑 바이어는 납기 SLA(도착일 ±1일 보장 + 지연 보상)를 계약서에 명시할 경우, 기존 공급처에서 전환할 의향이 높다.

| 항목 | 내용 |
|------|------|
| 검증 방법 | 도쿄 레스토랑 5~10곳 심층 인터뷰 + 샘플 제공 후 의향 조사 |
| 성공 기준 | 인터뷰 대상 70% 이상이 "납기 SLA가 전환 결정에 중요하다"고 응답 |
| 검증 기간 | 4주 (인터뷰 3주 + 분석 1주) |

---

### 가설 2: HACCP + 콜드체인 패키지가 선도 품질 우위를 입증 가능하다

**가설 내용**
HACCP 인증 완비 + IoT 온도 로그 제공 + 도착 불량 교환 보증 세트가, 도착 후 바이어의 품질 평가를 "기존 공급처 대비 우위"로 이끌 수 있다.

| 항목 | 내용 |
|------|------|
| 검증 방법 | 2kg 무료 샘플 발송 (5개 레스토랑), 도착 선도 평가 + 바이어 피드백 수집 |
| 성공 기준 | 샘플 수령 후 80% 이상이 "기존 공급처 대비 동등 또는 우위" 평가 |
| 검증 기간 | 3주 (샘플 발송 1주, 평가 1주, 피드백 수집 1주) |

---

### 가설 3: 일본어 전담 창구가 서류·소통 장벽을 충분히 해소한다

**가설 내용**
일본어 전담 담당자 + 통관 서류 원스톱 대행 서비스가, 바이어가 인식하는 "수입 행정 부담"을 실질적으로 제거하여 재주문으로 이어진다.

| 항목 | 내용 |
|------|------|
| 검증 방법 | 첫 납품 완료 후 바이어 인터뷰 (서류 처리 경험, 소통 만족도 평가) |
| 성공 기준 | 첫 납품 바이어 100% 재주문 의향 표명 (또는 3개월 내 재주문) |
| 검증 기간 | 6주 (첫 납품 + 3개월 추적) |

## 3. MVP 범위 최종 확정 (MoSCoW)

### Must (MVP 초기 릴리즈 필수 포함)

| 항목 | 이유 |
|------|------|
| HACCP 인증 완비 (국내) | 일본 수입 허가 전제 조건, 미보유 시 수출 불가 |
| 납기 SLA 계약서 | JTBD 핵심 성패 변수, OS=16 |
| 냉장 콜드체인 (출하~도착) | JTBD 핵심 성패 변수, 선도 보증 기반 |
| 일본어 전담 창구 (담당자 1인) | 바이어 Pain P3 직접 해소, 소통 장벽 제거 |
| 통관 서류 원스톱 대행 | OS=15, 바이어 전환 장벽 제거 |
| 실시간 물류 추적 (링크 제공) | OS=14, 납기 불안 해소 보조 수단 |
| 무료 샘플 프로그램 | 초기 바이어 유치, 리스크 없는 품질 검증 경로 |
| 도착 불량 전량 교환 보증 | 선도 SLA 신뢰성 담보 |

---

### Should (2차 릴리즈, MVP 이후 3~6개월)

| 항목 | 이유 |
|------|------|
| IoT 온도 로그 자동 리포트 | 콜드체인 추적 고도화, 현재는 수동 제공 가능 |
| 일본어 조리 가이드 (PDF) | 온보딩 품질 향상, 긴급 개발 불필요 |
| 월간 품질·납기 실적 보고서 | 바이어 관계 유지·신뢰 구축 |
| 발주 관리 웹 포털 (간이) | 이메일·LINE 대체, 발주 오류 감소 |
| 동남아 세그먼트 타진 | 일본 시장 안정 후 확장 |

---

### Could (3차 이후, 시장 검증 완료 후)

| 항목 | 이유 |
|------|------|
| 어종 다변화 (금태 외 프리미엄 어종) | 금태 단일 품목으로 먼저 검증 |
| 일본어 자동 번역 발주 시스템 | 규모 확장 시 필요 |
| 브랜드 스토리텔링 콘텐츠 (영상·SNS) | 인지도 구축, 초기 단계 불필요 |
| ASC 국제 양식 인증 | 동남아 확장 시 필요, 단기 우선순위 아님 |
| 바이어 커뮤니티 플랫폼 | 네트워크 규모 확장 후 검토 |

---

### Won't (현재 사업 범위 제외)

| 항목 | 제외 이유 |
|------|-----------|
| B2C 플랫폼 (일반 소비자 직판) | 타겟 세그먼트 외, 물류·CS 복잡도 급증 |
| 가공식품 개발 (통조림·훈제 등) | 다른 사업 영역, 제조업 역량 필요 |
| 자사 선박 운용 | 초기 CAPEX 과다, 물류 아웃소싱으로 대체 |
| 일본 현지 법인 설립 (단기) | MVP 검증 전 과도한 투자 |
| 가격 경쟁 전략 | JTBD 검증에서 기각된 가설 |

---

### MVP 초기 릴리즈 포함 범위 요약

```
MVP = HACCP 완비 + 납기 SLA 계약 + 냉장 콜드체인 + 일본어 전담 창구
    + 통관 서류 대행 + 물류 추적 링크 + 샘플 프로그램 + 도착 불량 교환 보증
```

**초기 릴리즈 목표**
- 기간: MVP 운영 시작 후 6개월
- 대상: 도쿄 오마카세 레스토랑 5개 이상과 공급 계약
- 성공 지표: 납기 준수율 95%+, 재주문율 70%+, 반품률 2% 미만

---

*작성 기준: 2주차 MoSCoW, JTBD, AOS/DOS OS 점수 분석 결과 반영*
